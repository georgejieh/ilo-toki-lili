# E15

Exploratory robustness items for visual domain shift.

The planned format reuses frozen scene JSON from earlier evaluation sets, then re-renders the pixels with noise, textures, partial occlusion, cluttered backgrounds, or other renderer variants. A small license-cleared human-drawn or photographed simple-scene set may be added later.

E15 is not pre-registered headline evidence. Its job is to show whether any grounded-model gap survives when the clean synthetic visual machinery is degraded.
