# Models

Model code lives here.

Planned modules:

- `blocks.py`: transformer building blocks.
- `a_text.py`: `toki-taso`, text-only baseline.
- `vision.py`: visual encoder initialized from scratch.
- `b_multimodal.py`: `toki-lukin`, visual-prefix transformer.
- `slots.py`: slot attention.
- `c_worldstate.py`: `toki-sona`, recurrent grounded model.
- `params.py`: parameter counts and parity checks.

No pretrained weights are used.
