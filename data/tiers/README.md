# Tiers

Dataset tiers stage the learning problem from concrete grounding to natural text.

- T0: word grounding.
- T1: compositional scenes.
- T2: episodes and before/after questions.
- T3: Toki Pona-only textbook lessons.
- T4: licensed community text.

Large generated corpora and imported community text stay in ignored local paths until a deliberate release process is used.
